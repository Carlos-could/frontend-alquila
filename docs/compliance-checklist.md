# Checklist de compliance mínimo (pre-lanzamiento)

Fecha: 2026-02-22
Estado: borrador

## Legal
- [ ] Términos y condiciones visibles y versionados.
- [ ] Política de privacidad visible y versionada.
- [ ] Consentimiento explícito para tratamiento de datos.

## Seguridad
- [ ] Revisión de permisos por rol completada.
- [ ] Políticas de acceso a documentos (Storage) aplicadas.
- [ ] Registro de auditoría para cambios críticos habilitado.

## Datos
- [ ] Política de retención definida por tipo de documento.
- [ ] Proceso de borrado/anonimización documentado.
- [ ] Acceso interno a datos sensibles restringido por rol.
